# Vivo — Landingpage

Dieses Mini-Template ist ein Vite + React Projekt für eine einfache Landingpage.

Schnellstart

Installiere Abhängigkeiten und starte den Dev-Server:

```bash
cd "Vivo Landingpage"
npm install
npm run dev
```

Build für Produktion:

```bash
npm run build
npm run preview
```

Dateien

- `index.html` — Einstieg
- `src/main.jsx` — React-Einstieg
- `src/App.jsx` — Hauptkomponente
- `src/components/Hero.jsx` — Hero-Bereich
- `src/styles.css` — Einfache Styles

Anpassungen

Passe `src/components/Hero.jsx` und `src/styles.css` an, um Look & Feel zu verändern.
